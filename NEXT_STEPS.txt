STEP 1: Setup Environment Variables
────────────────────────────────────
$ cp env.example .env
$ nano .env   (or use your favorite editor)

Add these values:
  - LEETCODE_SESSION     (from browser cookies)
  - LEETCODE_CSRF_TOKEN  (from browser cookies)
  - GITHUB_TOKEN         (from GitHub settings)

📖 See SETUP_GUIDE.md for detailed instructions!

STEP 2: Configure Usernames
────────────────────────────
$ nano config/config.yaml

Change:
  leetcode.username: "your_leetcode_username"
  github.username: "your_github_username"

STEP 3: Run the Sync!
──────────────────────
$ conda activate leetcode-project
$ python main.py

Follow the interactive prompts!
