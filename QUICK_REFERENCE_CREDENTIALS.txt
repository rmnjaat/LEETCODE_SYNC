┌──────────────────────────────────────────────────────────────────────────┐
│  📋 WHAT YOU NEED:                                                       │
│                                                                          │
│  1. LEETCODE_SESSION      (from browser cookies)                        │
│  2. LEETCODE_CSRF_TOKEN   (from browser cookies)                        │
│  3. GITHUB_TOKEN          (from GitHub settings)                        │
└──────────────────────────────────────────────────────────────────────────┘


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  🍪 LEETCODE COOKIES (Steps 1 & 2)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  🌐 Go to: https://leetcode.com (and login!)
  
  ⌨️  Press: F12
  
  📱 Chrome/Edge:
      Application tab → Storage → Cookies → https://leetcode.com
  
  📱 Firefox:
      Storage tab → Cookies → https://leetcode.com
  
  📋 Copy these cookie VALUES:
      • LEETCODE_SESSION     (long string, ~200-500 chars)
      • csrftoken            (shorter string)


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  🔐 GITHUB TOKEN (Step 3)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  🌐 Go to: https://github.com/settings/tokens
  
  🆕 Click: "Generate new token" → "Generate new token (classic)"
  
  ⚙️  Configure:
      Note: "LeetCode Sync"
      Expiration: 90 days
      Scopes: ✅ repo (check the box!)
  
  💾 Generate → Copy token immediately!
      Format: ghp_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
              (40 characters after ghp_)


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  💾 SAVE TO .env FILE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  📁 Create file:
     $ cp env.example .env
  
  ✏️  Edit file:
     $ nano .env
     
  📝 Paste your values:
  
     LEETCODE_SESSION=eyJ0eXAiOiJKV1Q...
     LEETCODE_CSRF_TOKEN=Bq8ks8TYPmR...
     GITHUB_TOKEN=ghp_AbCdEfGhIjKlMn...
  
  💾 Save and close!

  ❌ "LEETCODE_SESSION not set"
     → Check file is named .env (not env.example)
     → Check no extra spaces around values
  
  ❌ "Failed to authenticate"
     → Logout from LeetCode, login again
     → Get fresh cookies
  
  ❌ "GitHub connection failed"
     → Check token has 'repo' scope
     → Generate new token if expired
  
  ❌ "Can't find cookies"
     → Make sure you're logged into LeetCode
     → Try refreshing the page (F5)
